css eigenschappen

De afkorting CSS staat voor ‘Cascading Style Sheet’. Vrij vertaald betekent dat ‘Overkoepelend StijlDocument’.

De CSS-code voor de opmaak staat in dit losse CSS-bestand en is geldig voor iedere HTML-pagina van je website. Hierdoor is direct het probleem opgelost, dat vroeger ontstond als je op vijfhonderd pagina’s de achtergrondkleur moest aanpassen. Maak je gebruik van een CSS-bestand voor de opmaak, dan hoef je de code maar op één plek aan te passen. De wijziging is dan geldig voor alle pagina’s van je website.

Met de volgende regel maak je de koppeling met het CSS-bestand. <link rel="stylesheet" type="text/css" href="style.css">

Het head-element ziet er dan als volgt uit:

<head>

<>title>Werken met CSS</title>

<link rel="stylesheet" type="text/css" href="style.css">

</head>

voorbeelden van een tag-selector in een css bestand.

    • text-align:center;
    • font-family:Monospace;
    • p{style="list-style-type:none;">
    • color: #000000;
